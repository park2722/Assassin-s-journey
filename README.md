# AR Encounter-and-Capture_Creature(가제)

## 📖 프로젝트 개요
본 프로젝트는 **컴퓨터 비전(Computer Vision) 기술과 제스처 인식**을 활용하여 구현한 증강현실(AR) 기반 상호작용형 몬스터 배틀 프로그램입니다. 
체스보드를 마커로 사용하여 현실 공간 위에 3D 몬스터와 수풀을 렌더링하고, 웹캠을 통한 사용자의 손가락 방향과 제스처(손바닥 펴기, 팔 휘두르기 등)를 실시간으로 인식하여 캐릭터의 이동, 전투, 도망, 포획 등의 액션을 구현할 예정입니다.

## 🧠 핵심 컴퓨터 비전 알고리즘 및 구현 로직

### 1. 3D 공간 투영 및 카메라 포즈 추정 (Camera Pose Estimation)
휴대폰의 카메라와 연동하여 체스보드를 확인하고 그 위에 3D 에셋을 띄우는 방식으로 진행할 예정입니다.

### 2. 핸드 랜드마크 기반 3D 제스처 상호작용
* **방향 및 이동 제어**: 컴퓨터의 카메라를 활용하여 사람의 제스처를 인식해 캐릭터의 움직임, 상호작용을 구현할 예정입니다.
* **상태 머신 (State Machine) 기반 전투 시스템**:
  - `선택지 조작`: 화면상의 [전투 / 도망 / 포획] UI를 손가락으로 가리켜 선택
  - `공격 (전투)`: 손바닥을 펴고 앞으로 뻗는 제스처를 인식해 공격 이벤트 발생
  - `포획`: 팔을 휘두르는 모션 인식 시, 포획물이 날아가 크리쳐 포획

### 3. 정적 3D 에셋 렌더링 및 수학적 변환 (Transformations) - 가능하다면 추가사항
랜더링된 3D 에셋을 기반으로 간단한 형태의 모션을 구현
